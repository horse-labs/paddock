# decisions/

Decisões arquiteturais (ADRs) — o porquê das escolhas estruturais. Um arquivo por decisão:
`NNNN-<slug>.md` (ex.: `0001-banco-de-dados.md`).

Se seus ADRs vivem em outro lugar (repo de docs, wiki), use esta pasta só como **índice linkado** —
não duplique. O importante é que a decisão tenha um ID estável citável no `status.md`.
